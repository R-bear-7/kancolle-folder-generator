## Usage (Windows)

### Recommended
1. Download `kancolle-folder-generator.exe`
2. Double click the file
3. Select a folder
4. Enter the number of event stages
5. Done!

## Note

Windows may show a security warning when running the `.exe` file.

If that happens:
- Click "More info"
- Then click "Run anyway"

This is normal for unsigned applications.
